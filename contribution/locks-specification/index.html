Signature

The locking mechanism used by Rosmaro has the following signature:

lock = async () => unlock

Calling the lock function acquires a pessimistic lock. It takes no arguments and returns a Promise that resolves to the unlock function. Calling the unlock function returns a Promise which if resolved indicates that the previously acquired lock has been released.

Errors

Both lock and unlock may throw errors. It's very important those errors are Error objects. Throwing something else may cause a crash.
